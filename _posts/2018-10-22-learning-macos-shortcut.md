## 切换
```
ctrl+left／right  全屏之间的来回切换
command + space  打开Spotlight Search
ctrl + space  中英文输入法切换
```

## 截屏工具
```
shift + command + 4  保存到桌面
ctrl + shift + command + 4  保存到剪贴板
```

## 截全屏工具
```
shift + command + 3  保存到桌面
ctrl + shift + command + 3  保存到剪贴板
```

## 特殊字符输出
```
option + 8 在英文输入状态下，输出英文名字中间的点号
```
